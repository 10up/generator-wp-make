# generator-wp-make

> [Yeoman](http://yeoman.io) generator


## Getting Started

If you need it, install Yeoman through npm:

```
$ npm install -g yo
```

### WP Make

To install generator-wp-make clone this repository and run:

```
$ npm link
```

Finally, in the desired directory, initiate the generator:

```
$ yo wp-make
```
